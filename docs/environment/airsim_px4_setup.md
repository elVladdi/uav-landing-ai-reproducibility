# AirSimNH/PX4 Setup

Use this file as the public setup entry point. The original experiment used AirSimNH with PX4 SITL in WSL2 and a MAVLink direct UDP channel for the Python control scripts.

Public templates:

- `configs/airsim_px4_settings.example.json`
- `configs/px4_airsim.env.example`

Machine-specific IP addresses, simulator install paths and private `.env` files must remain outside Git.
