# Simulator Startup Manual

Minimal startup order:

1. Start AirSimNH with the public settings template adapted locally.
2. Start PX4 SITL in WSL2.
3. Start the MAVLink route for the Python agent.
4. Confirm local position and bottom camera image availability.
5. Run a short takeoff/land test before any formal run.

The detailed project manual remains in `docs/fase02_entorno_experimental/09_manual_arranque_entorno_simulado.md`.
