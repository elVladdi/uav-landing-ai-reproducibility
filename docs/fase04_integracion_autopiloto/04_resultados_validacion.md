## Criterio de aceptación

Una corrida no debe aceptarse como positiva si:

- no hay marcador visible en `bottom_center`;
- el detector marca una region que no es la plataforma;
- PX4 no reporta posicion local valida;
- Offboard no inicia correctamente;
- los comandos visuales tienen signo incorrecto;
- se pierde el marcador y el sistema no aborta o aterriza;
- falta el log integrado de telemetría, detección y comando.
