# MDPI-Style Reproducibility Statement

The experiment was designed as a controlled and reproducible AirSim-PX4 simulation study. The public repository provides the configuration snapshot, curated run-level dataset, analysis scripts, final derived tables and checksum manifest required to reproduce the analytical results reported for the T0/T1 comparison.

Full raw logs and simulator artifacts are excluded from Git because of size and environment specificity. Their inventory and availability notes are documented separately.
