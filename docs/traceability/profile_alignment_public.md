# Public Profile Alignment

The reproducibility package aligns with the project objective: develop and evaluate a reproducible AirSim-PX4 framework for autonomous UAV landing assisted by Offboard vision over a signaled platform in controlled simulation.

| Profile element | Public artifact |
|---|---|
| Architecture | `docs/fase01_*`, `docs/fase04_*`, `src/` |
| Visual perception | `src/perception/`, `configs/perception_config.json` |
| Offboard/PX4 integration | `src/control/`, `configs/control_config.json` |
| Reproducible protocol | `docs/methodology/`, `configs/phase05_experiment_config.json` |
| Metrics and analysis | `outputs/tables/phase05_experiments/`, `outputs/tables/phase06_analysis/` |
