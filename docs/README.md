# Documentation

This public reproducibility package keeps the documentation needed to understand and reproduce the AirSim-PX4 UAV landing experiment.

## Main sections

- `methodology/`: public experimental design, curation rules, statistical plan and scope.
- `environment/`: AirSimNH/PX4 setup notes and simulator startup guidance.
- `traceability/`: alignment with objectives and documented protocol amendments.
- `article_support/`: MDPI-style reproducibility and data availability text.
- `fase05_experimentacion_t0_t1/`: formal T0/T1 execution protocol and curation documentation.
- `fase06_analisis_resultados/`: analysis, discussion, limitations and conclusions.

Binary thesis drafts, downloaded papers, bibliography spreadsheets and private working documents are intentionally excluded from the public package.
