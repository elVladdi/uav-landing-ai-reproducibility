# Raw Logs Availability

The full raw CSV logs are not included in this public Git branch. This folder provides an inventory derived from the curated Phase 05 summary.

- `raw_logs_inventory.csv`: run-level references to the original raw log paths.
- `raw_logs_checksums.sha256`: placeholder manifest indicating that raw logs are not included in public Git.

If the raw archive is deposited externally, replace the placeholder entries with real SHA256 hashes and add the DOI or archive URL here.
